# Story Game App

A dynamic, coin-based story game where players progress through chapters using coins. Chapters are fully downloadable and updated dynamically, allowing players to enjoy new content without requiring app updates.

## Features
- Coin-based chapter progression with automatic refill every 1.5 hours  
- Dynamic story updates using Firebase JSON  
- Assets downloaded locally using ASD Downloader extension  
- Lightweight login/signup using Google Sheets (tracks user progress & coins)  
- Push notifications for story updates via OneSignal  
- AI-generated images replacing copyrighted content  
- Planned monetization with ads and Play Store release  

## Tech Stack
- **Niotron** – App development  
- **Firebase** – JSON & assets storage  
- **Google Sheets** – User data management  
- **ASD Downloader** – Asset management  
- **OneSignal** – Notifications  
- **Stable Diffusion** – AI-generated images
